Целью данной работы является изучение гридов и программного обеспечения промежуточного уровня класса DRMS-систем Sun N1 Grid Engine 6.0 для организации гридов. В ходе выполнения работы была развернута грид-система и разработано приложение, выполняющее прикладную задачу на ней с помощью программного интерфейса DRMAA.

Исходный код в формате проектов для NetBeans (в архиве).
